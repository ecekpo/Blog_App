# 📖 Blog App <a name="about-project"></a>


**Blog App** The Blog app will be a classic example of a blog website. This functional website will show the list of posts and empower readers to interact with them by adding comments and liking posts.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Ruby and Rails

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Create a Posts.**
- **Like a Post.**
- **Comment on a Post.**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- Ruby installed on your machine.
- gem install rails

### Setup

Clone this repository to your desired folder: 

`git clone https://github.com/ecekpo/blog_app.git`


### Install

Then install the project's necessary dependencies with: 

`bundle install`

### Usage

To run the project, execute the following command:

`rails server`

### Run tests

To run tests, run the following command:

`rspec`
